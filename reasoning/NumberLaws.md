# 数字规律

## 例题

### 前项相加（斐波那契数列）

A1 + A2 = A3

[ 1, 1, 2, 3, 5, 8, 13, 21, 34 ]


### 位数相加

1+7 = 3+5 = 2+6 ...

[ 17, 35, 26, 44]


### 递推

前一项除2 加 当前项 为 后一项
24/2 + 14 = 26, 14/2 + 26 = 33, 26/2 + 33 = 46

[ 24, 14, 26, 33, 46 ]

### 分子分母交叉规律

1/3, 1/2, 2/3, 3/4, 8/9, 27/32, ?

前一项分子 X 当前项分母 = 后一项分子
前一项分母 X 当前项分子 = 后一项分母
